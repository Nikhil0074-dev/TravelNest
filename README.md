# TravelNest – Trip Planning Platform

A modern, responsive travel planning website that helps users discover popular destinations and access useful travel planning tools.

##  Preview

TravelNest is a single-page website featuring:
- **Header** - Logo, navigation, and "Plan Trip" CTA button
- **Hero Section** - Search functionality for destinations
- **Popular Destinations** - Curated travel spots (Paris, Bali, New York)
- **Planning Tools** - Itinerary Builder, Budget Planner, Route Mapping, Accommodation Checklist
- **Travel Tips** - Essential advice for travelers
- **Call-to-Action** - Promotional section
- **Footer** - Copyright information

##  Project Structure

```
TravelNest/
├── index.html    # Main HTML structure
├── style.css     # Styling and animations
├── script.js     # Interactive functionality
└── README.md     # Project documentation
```

##  Getting Started

1. Clone or download the project
2. Open `index.html` in any web browser
3. No build steps or dependencies required

##  Features

- **Responsive Design** - Works on mobile, tablet, and desktop
- **Interactive Navigation** - Mobile-friendly navbar with toggle
- **Sticky Header** - Header becomes fixed on scroll
- **Scroll to Top** - Quick navigation button
- **Card Hover Effects** - Smooth animations on destination cards
- **Search Functionality** - Input field for destination search
- **External Images** - Beautiful travel photos from Unsplash
